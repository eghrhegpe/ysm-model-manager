模型仓库页面（app-tree）设计规则：

1. **页面分区**：顶部工具栏（搜索+批量操作）→ 主内容区（嵌套树）→ 底部操作栏（统计+操作按钮+预览切换）

2. **文件夹图标**：▶未展开/▼已展开。可根据内部文件状态着色（全启用绿色、全禁用灰色、混合橙色）

3. **文件图标**：按扩展名着色——.ysm 紫色/蓝色、.zip/.rar 黄色、图片类渲染缩略图、文件夹统一黄色

4. **_disabled 特殊处理**：以 `_` 开头的文件夹文字 Opacity 0.5，前面加 🔒 标识。点击"全部启用"时忽略此文件夹

5. **搜索即时过滤**：同时过滤文件和文件夹名。匹配到的文件夹自动展开并高亮关键词

6. **批量操作联动**：当有文件被选中（复选框勾选）时，"全部启用/禁用"按钮只作用于选中项。操作后文件图标应闪烁变色反馈

7. **去重逻辑**：基于 MD5 Hash 或文件大小比对。触发后弹模态框列出重复组，让用户勾选保留哪个，不可直接删除

8. **预览面板**：右下角"◀预览"按钮切换侧栏。展开时显示选中文件的详细信息（大图/版本/作者/描述/依赖项）

9. **数据结构**：每个条目包含 id, name, type (folder|file), path, isExpanded, children, status, extension, size, modified, isEnabled, hash, thumbnail
