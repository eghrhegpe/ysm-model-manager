## v1.0.2 — 导入/重命名 UX 大修 + 弹窗统一

### ✨ 新功能
- **多文件拖拽导入**：支持文件夹递归扫描，自动队列进料
- **批量重命名对话框**：统一作者/作品，勾选文件，实时预览
- **文件名解析增强**：支持 `[[双括号]]` author、`《书名号》` work

### 🐛 修复
- 文件夹右键"打开所在文件夹"无反应
- 文件夹右键"新建子文件夹"路径错误
- `window.showConfirm` 确认框全局静默失效
- 导入文件已存在时覆盖确认不弹窗
- `.lnk` 快捷方式拖入崩溃

### 🎨 优化
- 所有弹窗统一为卡片式模态风格
- 单文件重命名对话框宽度 400px → 640px
- 重命名后树自动刷新

### 🧹 清理
- 删除重复解析器、死代码
- 旧版 `prompt()` 标为 @deprecated
