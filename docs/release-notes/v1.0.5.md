# v1.0.5 — P5 功能闭环版

> 发布日期: 2025-06-07

## 🎯 核心改进

### ⬇️ 下载系统全面重构

- **三入口统一**：单击/多选/全选下载统一走 `enqueueDownloadTasks`，只注册一组 Wails 事件
- **进度防骗**：大文件秒跳 100% 时锁定 99%，加旋转菊花动画；小文件极速通道 150ms 过渡
- **错误聚合**：下载失败集中显示摘要，进度条变红，不再逐条弹框
- **心跳保底**：每 256KB 或每 200ms 强制汇报进度，Content-Length=-1 时仍可显示

### 📁 文件夹导入保持目录结构

- 新增 `ImportModelFileTo(name, subpath, base64)` Go Binding
- 拖入 `folder/sub/model.ysm` 时在仓库中保持 `folder/sub/` 目录结构
- 导入对话框显示子路径指示 `📂 folder/sub/`

### 🎨 模型预览图

- 文件名映射：`.ysm` → `.png`/`.jpg`/`.webp` 自动推导
- Hover 浮动预览 + 预加载 + 三后缀降级
- 淡入动画（opacity + scale）

### 🏷️ 作者标签

- 仓库页搜索框下方显示作者标签栏
- 点击按 `[作者名]` 筛选，再次点击取消

### 🔢 统计数字动效

- 里程表逐位滚动动画（个位→十位→百位）

### 🔀 树组件多选

- Ctrl+点击不连续选择，Shift+点击范围选择
- 右键批量菜单：全部启用/禁用、批量重命名、移入回收站
- 底部 stat 显示「已选 N 个文件」

### 🛡️ 防御性架构

- 独立模块防误删（`authors.js` 从 `index.js` 分离）
- `connectedCallback` try/catch 兜底，初始化异常不白屏
- 预览图共享单例，防止 DOM 节点泄漏
- 仓库路径变更实时重载（`tree:reload` 事件）

### 🔧 其他

- 设置页改仓库路径后自动刷新树
- 统计数字动效改为里程表逐位滚动
