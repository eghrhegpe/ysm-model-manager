- YSMParser.exe **不再打包进发版 zip**（WASM 已内嵌）

## 📋 Debug 记录

8 轮调试详记于 `docs/postmortem-20250608-wasm.md`：

1. MEMFS callMain I/O 不可靠 → 内存解析 bridge
2. 新增 C++ bridge 暴露内存解析接口
3. HEAPU8 闭包变量 → 胶水代码注入 patch
4. `const` 重赋值 → `let`
5. `.replace()` 歧义匹配函数定义 → 带分号前缀
6. ALLOW_MEMORY_GROWTH 后 HEAPU8 分离 → malloc 后重取
7. `new Uint8Array(string)` 在 WebView2 返回空 → `Uint8Array.from()`
8. `[]byte` 经 JSON 序列化为 base64 → `atob()` 解码
