## 🎯 核心改动

| `workshop-events.js`      | ~15 处 inline | 零 inline         |

新建 CSS 类前缀体系：`ysm-`（预览）、`ws-`（创意工坊）、`dlg-`（对话框）、`br-`（批量重命名）、`repo-`（仓库）、`stg-`（设置）

> ⚠️ 修复：Shadow DOM 组件（`app-content`）的 CSS 类必须写在 `content-css.js`，全局 `components.css` 无法穿透。

### 🗃️ 持久化缓存

- `_previewDataCache` + `_ysmCache` 从组件实例级 → 模块级 `Map`（`utils/preview-cache.js`）
- 切换页面不再重复解码 .ysm 文件

### 📋 批量导出骨骼结构

- 仓库工具栏新增「📋 骨骼结构」按钮
- Go `ExportBoneStructures()` 扫描所有模型，输出格式化文本报告
- 包含文件名、骨骼名、立方体数、纹理尺寸、统计汇总

### 🧭 MC 目录智能检测

- `isLikelyMinecraftDir()` — 内容级检测（`versions/`、`assets/`、`launcher_profiles.json`）
- `scanMinecraftDirs()` — 扫描 10+ 常见位置（PCL2、%APPDATA% 等）
- `ValidateMinecraftDir()` — 选父目录自动补 `.minecraft` 子目录
- 首次启动自动检测并设为默认
- 设置页「自动搜索」按钮改用新 API

### 🧩 代码拆分

- `index.js`（app-preview）：**570 → 470 行**，抽出 `data.js`、`tpl.js`、`events.js`、`utils.js`
- `disconnectedCallback` 新增 `_cleanupModelListeners()` 防内存泄漏
- 调试日志 16 行 → `devLog()`，仅 DEV 模式输出

### 💡 空状态优化

- 设置页路径未设置时显示 `💡 请点击...` 提示，设置后自动隐藏
- 侧栏搜索无结果显示 `🔍 未找到匹配的整合包`
- 仓库路径新增「🔍 自动搜索」按钮，扫描桌面/文档/下载等常见位置

### 🔧 工程改进

- `.gitignore` 新增 `build/bin/`、`build/release/`、`*.exe`、`node_modules/`
- `constants.js` 创建（Canvas 尺寸/缩放/动画常量）
- `build-release.ps1` 支持 GitHub Release 自动上传（gh CLI / GH_TOKEN）
- 版本号统一：`v1.1.0` 和 `1.1.0` 都兼容
