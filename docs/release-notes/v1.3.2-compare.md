# v1.3.2 — 开发者对比（技术细节）

## 🔧 `SaveAppConfig` 先写文件再校验（实现细节）

之前 `SaveAppConfig` 先对 mcRoot 执行 `ValidateMinecraftDir` 校验，校验失败就返回错误，**配置文件不会被创建**。这导致：

- 侧栏/设置页设置 mcRoot 后配置文件可能不生成
- 只有设 repoRoot（跳过校验）才能创建文件
- 如果 `ysm_config.json` 被删除，重启后无法自动恢复

**修复**：先 `os.WriteFile` 写入配置文件，再校验。校验失败只是不启动 watcher，不影响文件存在。
