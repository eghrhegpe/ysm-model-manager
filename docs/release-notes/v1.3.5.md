# v1.3.5 — WASM 性能优化 + 内存泄漏修复 + mcRoot 配置修复

> 发布日期: 2026-06-10

## 🐞 Bug 修复

### 修复 `SaveAppConfig` 未保存修正后的 mcRoot 路径

v1.3.4 将 `SaveAppConfig` 改为"先写文件再校验"，但写入的是原始路径而非校验后的修正路径，导致下次启动读取到错误路径。

**修复**：先执行 `ValidateMinecraftDir` 拿到修正路径，再用修正路径写配置文件和启动 watcher。

### 🚀 WASM 解码器性能优化

- **base64 解码加速** — `Uint8Array.from` → `for` 循环，约 3-5x 提速
- **WASM 二进制缓存** — 首次解码后缓存，二次调用零开销
- **胶水代码缓存** — 同
- **脚本注入消除** — `<script>` DOM 注入 → `(0, eval)`，约 5x 提速
- **移除 callMain 回退** — 跳过必定失败的 callMain，节省 ~100ms
- **`ReadFileBytes` base64 加速** — 同 `for` 循环优化

### 🧠 内存泄漏修复（模型越加载越卡）

- **缓存无限增长** — 上限 50 条，FIFO 淘汰
- **Blob URL 只创建不释放** — 淘汰时自动 `URL.revokeObjectURL()`
- **切换模型旧 DOM 堆积** — 加载前清空前一模型内容
- **3D 僵尸循环** — 关闭时彻底取消动画循环 + dispose controls + 清理 resize 监听
