## 实现细节

### 导航重构
```
v1.4.0: 📦 | 🎮 | 🧩 创意工坊 | 🛠️(！) | ⚙️
v1.4.1: 📦 | 🎮 | 🎨 创作者频道 | 🧩 创意工坊 | 👴 仓库元老 | 🛠️ | ⚙️
```

### 仓库页布局
之前每个 tab 独立布局，导入/回收站/去重 tab 的右侧是空的或自定义内容。现在：
```
┌─ tab 栏 ─────────────────────────────────┐
│ [📁] [📥] [🗑️] [🔗]                      │
├──────────────┬────────────────────────────┤
│ 左面板(自适应) │ 右栏 app-preview (220px)  │
│ (切换 tab 不变)│ (统一显示模型详情)          │
└──────────────┴────────────────────────────┘
```

### 创意工坊数据流
`workshop_gitHub.json` → Go `LoadGitHubRepos()` → 前端 `_initGithub()` 直接加载，不再从 `loadWorkshopData()` 过滤。
