# v1.4.1 — 导航重构 + 多资源架构奠基

> 发布日期: 2026-06-10

## ✨ 新功能

### 🧩 创意工坊独立为导航页面

GitHub 仓库浏览从原来的「创意工坊」中分离，成为独立的一级导航页「🧩 创意工坊」。使用独立的配置文件 `workshop_gitHub.json`。

### 🎨 创作者频道

原「创意工坊」更名为「🎨 创作者频道」，创作者数据文件从 `workshop_creators.json` 迁移为 `creators.json`。

### 👴 仓库元老晋升主菜单

「仓库元老」（含仓库健康度评分、最早模型展示、今日推荐）从诊断页的子功能升级为独立导航项。

### 🖼️ 仓库页共享预览面板

四个 tab（文件树 / 导入 / 回收站 / 去重）现在共享右侧 220px 的模型详情预览面板，布局统一。

## 🔧 改进

- 导航栏宽度从 200px 缩减至 160px，移除诊断页 `!` 标记
- GitHub 仓库加载失败时显示「↗ 在 GitHub 中打开」按钮
- 诊断页移除「模型去重」（功能保留在仓库页）
- 模型预览底部的「导出 PNG」按钮移除，3D 按钮移至与「导出骨骼名」同行
- 仓库页 tab 切换 ID 修复（`versions` → `tree`）
- 修复 `githubHTML()` 缺少 return 语句导致页面崩溃的问题

## 📄 配置文件变更

| 旧文件 | 新文件 | 说明 |
|---|---|---|
| `workshop_creators.json` | `creators.json` | 创作者频道数据 |
| — | `workshop_gitHub.json` | 创意工坊 GitHub 仓库配置（新建） |
| `workshop_sites.json` | 不变 | 已移除 GitHub 站点条目 |

## 🗑️ 移除

- 诊断页的「模型去重」按钮和面板
- 模型预览「导出 PNG」按钮
- `workshop_sites.json` 中的 GitHub 站点
