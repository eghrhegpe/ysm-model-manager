## 架构变更

| 层级 | 变更 | 影响 |
|------|------|------|
| Go 后端 | 新增 `go/ysm/texsize.go` | 纹理尺寸扫描不依赖 YSMParser/WASM，大型仓库秒出 |
| Go 后端 | `exec.Command` + `HideWindow` | CLI fallback 不再弹控制台窗口 |
| 前端 3D | `_toggle3D` 替代 `viewBtn` | 无隐藏元素无轮询，🎨 3D 标签直接触发全屏渲染 |
| 前端 WASM | V3 MEMFS 路径 | 42 个 V3 文件可解码（之前全部回退 Go CLI） |
| 前端 WASM | `forceVer` + V3 偏移 | `buildStdYsgpFromTextVariant` 可尝试 V2/V3 两种重建策略 |
