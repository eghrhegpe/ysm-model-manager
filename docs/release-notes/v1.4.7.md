# v1.4.7 — V3 WASM 解码 + CLI 无窗 + 3D 标签页整合

> 发布日期: 2026-06-12

## ✨ 新特性

### V3 文本头部 WASM 解码

- 新增 `detectYsmVersion` 预检 → V3 走 MEMFS `decodeYsmFile` 路径（支持文本头部）
- `buildStdYsgpFromTextVariant` 加 `forceVer` 参数，V2 解码失败后自动尝试 V3 头部重建
- 42 个 V3 文件测试通过，最高 306 骨骼 / ~4.7MB 模型
- `_wasmTried` 缓存标记，防止重复尝试 WASM 解码

### 轻量级纹理尺寸扫描

- 新增 `GetModelTexSizes` Wails Binding（`go/ysm/texsize.go`）
- 不调用 YSMParser/WASM，直接从 ZIP/7z JSON 提取纹理尺寸
- 筛选面板「仅纹理筛选」走此轻量路径，大型仓库秒出结果

### 3D 标签页整合

- 标签栏新增独立的 🎨 3D 按钮，直接触发全屏 3D 预览
- 骨架 tab 中原有的 🌐 3D 按钮已移除（功能合并到 🎨 3D）
- `_toggle3D` 替代 `viewBtn.onclick`，消除隐藏元素和轮询

### CLI fallback 无窗运行

- YSMParser.exe 两个调用点均添加 `HideWindow: true`
- 不再弹出黑色控制台窗口闪烁

## 🐛 Bug 修复

### V3 二进制段偏移修正

- V3 格式的加密数据前**没有 16B 独立 hash 区域**（V2 有）
- `buildStdYsgpFromTextVariant` 根据版本号判断偏移量，不再硬编码跳过 16B

### 重复 WASM 解码

- `_loadPreviewImage` 和 `_loadModel2D` 都可能触发 WASM 解码，之前模型会解码两次
- `_wasmTried` 缓存标记确保只尝试一次

### V1 模型 JSON 路径

- 根目录级 JSON 文件（无 `models/` 前缀）也被正确识别

### WASM 胶水代码补丁

- `ysm-parser.js` 中 `replace` 改为 `replaceAll`，确保所有 `updateMemoryViews()` 调用点都被正确 patch

### CSS 变量替换

- 诊断面板多处硬编码 `#6c7086` / `font-size:9px` 替换为 CSS 变量

## 🔧 重构

- `viewBtn`/`viewHint` DOM 元素移除，3D 全屏逻辑移至 `_toggle3D` 函数
- `close3D()` 中的按钮文字重置代码已清理
- `switchTab` 闭包解除（`_loadModel2D` 不再依赖 `_showModelDetail` 的局部函数）
