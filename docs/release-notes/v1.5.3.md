# v1.5.3 — 界面翻新 & 多资源统一

> 发布日期: 2026-06-13

---

## ✨ 新特性

### 界面文字全面调大

全局字号 +1px，正文 12px、按钮 11-12px、标签 12-13px。导航栏文字更大一号（15px），一眼看清。

### 设置页支持基准字号调节

「设置 → 界面与体验 → 基准字号」可选 **小/标准/大**，按钮间距跟随缩放，切换即时生效。

### 🎨 材质包 / ☀️ 光影包 / ⚙️ 蓝图 统一界面

现在所有资源类型（YSM、MMD、VRC、材质包、光影包、蓝图）使用**相同的文件树 + 预览面板界面**，操作一致，不再有割裂感。

### YSM json / MMD 模型纹理安全同步

推送模型到整合包时，同一文件夹内的纹理文件（`.png`、`.tga` 等）会**一起复制**，不再缺失纹理。

### 窗口位置记忆优化

双显示器用户关闭再打开应用后，窗口不会跑到屏幕外。如果显示器布局变了，自动用相对比例重算位置。

## 🔧 改进

### 界面一致性

- 所有按钮/tab 按功能角色统一字号和间距（导航、标签、筛选、主要按钮、行内按钮、工具栏按钮）
- 设置页路径卡片的操作按钮（🔍 自动搜索 / ↩️ 默认）移至标题栏右侧
- 「链接模式」的「重新应用链接」按钮从底部移到标题行
- 侧栏导航新增「🧭 导航栏」分区标识
- 路径默认提示改用 Windows 反斜杠风格

### 光影包详情支持

- 新增 `ReadShaderpackLang` 读取光影包 ZIP 内 `lang/en_US.lang`
- 解析配置条目，光影包详情页不再空白（显示配置项数量或描述）

### 内部优化

- 窗口位置合并到 `ysm_config.json`，根目录少一个文件
- 弹出通知（Toast）颜色适配所有主题（浅色/深色）
- 右键菜单颜色适配所有主题
- 设置页面包屑不再被 `overflow: hidden` 裁剪
- `GetResourceInstanceStatus` 和 `InstallResourceToInstance` 中硬编码的 subDir switch 全部替换为 `types.SubDirMap()`，新增类型时无需改多处
- `bug-chronicle.md` 新增 #32 (正则空格) / #33 (括号不匹配) 两条排查记录
