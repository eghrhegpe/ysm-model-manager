# v1.5.7 — PrismLauncher 支持 & 性能优化 🚀

## 变更内容

### 社区贡献合并

合并了 @zuogeren1 的 PR #2，包含 PrismLauncher 实例布局支持及多项性能优化。由于 PR 分支从 v1.5.4 分叉（未包含 v1.5.5 的 CSS 变量化），未直接 merge，而是手动 cherry-pick 关键改动到当前 main。

### 影响范围

- **重大**：`go/sync/sync.go` — `ListVersions` 重构为三布局自动检测。`go/ysm/ysm.go` — `HasYSMMod`/`HasModInDir` 文件名预过滤。
- **中度**：`app_config.go` — PrismLauncher 路径自动探测。`app_scan.go` — scanCacheTTL 30s。前端 sidebar 并发锁 + 防抖 + 延迟构建。
- **轻微**：设置页鸣谢、提示修正、README 更新。
- **无**：API 破坏性变更。
