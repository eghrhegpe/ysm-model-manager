# v1.5.8 — 创意工坊修复 & 自动扫描优化 🔧

## 变更内容

### PR #3 合并（zuogeren1）

手动应用了 @zuogeren1 的 PR #3 修复，包含 6 个文件的改动。

重写了 `scanMinecraftDirs()`，从硬编码 C/D/E 盘特定路径改为自动扫描 C-Z 所有可用磁盘，覆盖 PrismLauncher/MultiMC/PolyMC 安装版路径和嵌套目录。

hover 显示扫描范围和结果列表、多路径时弹出选择器。

- **严重修复**：`app-modules.js` 动态导入变量 → 字面量。Vite 构建从 25 modules 恢复至 98 modules。

- **功能改进**：`app_config.go` 全盘自动扫描。`workshop-settings.js` hover tooltip + 多路径选择。
- **UI 修复**：仓库页路径按钮导向设置页。CSS 补色/布局修复。
